# info-geo

**Information geometry for agent state spaces** — Fisher information matrices, natural gradients, and Riemannian structure over agent parameter manifolds.

## What This Gives You

- **Fisher information** — compute the Fisher information matrix for agent distributions
- **Natural gradient** — Riemannian gradient descent on parameter manifolds
- **KL divergence** — measure distances between agent state distributions
- **Geodesic computation** — shortest paths on the statistical manifold

## How It Fits

Mathematical foundation for agent optimization in the SuperInstance fleet. Provides the geometric framework that `tropical-attention` and `pareto-tournament` build on.
